## üöÄ About the Game

CRT Spaceship is a retro-inspired space shooter that recreates the aesthetic of classic CRT terminal displays. Built with pure HTML5 Canvas and Web Audio API, it features:

- **Authentic CRT Effects**: Scanlines, vignette, phosphor glow
- **Procedural Audio**: All sounds generated in real-time
- **Smooth Performance**: Optimized for 60 FPS gameplay
- **Mobile Support**: Touch controls with gesture handling
- **No Dependencies**: Pure vanilla JavaScript
