# Debug Reports Directory

This directory contains automatically generated debug analysis reports.

## Contents

- `debug-report.md` - Latest debug analysis report (auto-generated every 3 days)

## Report Contents

Each report includes:

1. **Telemetry Statistics**
   - Total sessions analyzed
   - Error counts
   - Performance issue counts

2. **Distribution Analysis**
   - Browser distribution
   - Device distribution
   - Operating system distribution
   - Network quality distribution

3. **Issues Detected**
   - Error logs with severity
   - Performance issues
   - Occurrence counts
   - Status tracking

4. **Recommendations**
   - Optimization suggestions
   - Bug fix priorities
   - Device-specific improvements

## Generation

Reports are automatically generated by the `game-update.yml` workflow which runs every 3 days.
